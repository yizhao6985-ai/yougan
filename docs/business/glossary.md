# 术语表

| 术语 | 代码 / 字段 | 说明 |
|------|-------------|------|
| 灵感模式 | `inspiration` | 收集并确认创作需求 |
| 创作模式 | `creation` | 创意总监制定制作计划并出稿 |
| 提问模式 | `ask` | 自由答疑与创作咨询 |
| 创作脉络 | Creative Context Panel | Studio 右侧栏：灵感、制作计划、预览、参考素材 |
| 制作计划 | `Work.plan` / `WorkProductionPlan` | 待执行与已执行的制作任务集合 |
| 已定稿计划 | `plan_ready` | 制作计划确认完成，制作团队可执行 |
| 成稿 | `Work.creation` / `GeneratedContent` | 标题、正文、标签等最终产出 |
| 作品 | `Work` | 跨对话共享的长记忆容器 |
| 对话 | `WorkConversation` | 作品下的单条聊天线，独立 `threadId` |
