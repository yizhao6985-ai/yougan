## 计费单位：「AI 创作」

### 计量规则

- 每次经 API `/langgraph` 代理的 Agent 请求成功前，调用 `consumeAiUsage`。
- **每计 1 次**，与对话轮数基本 1:1（一次用户发送 → 一次扣费）。
- 超额时返回错误，前端提示「本月额度已用完，升级 Pro 或等待下月重置」。

### 周期重置

| 套餐 | 周期 |
|------|------|
| 免费版 | 自然月（UTC 月首重置 `aiUsageThisPeriod`） |
| Pro | 订阅周期（月付/年付对应 `currentPeriodStart/End`） |

### 商业含义

- 额度与 **LLM API 成本** 直接挂钩，便于毛利测算。
- 高频创作者（日更）会自然触达 30 次上限，形成升级压力。
- Pro 500 次/月 ≈ 日均 16 次，适合日更或多作品并行用户。

## 订单与支付流程

### 当前实现（模拟支付）

1. 用户在 `/settings/membership` 选择 Pro + 月付/年付。
2. `checkoutBillingOrder` 创建 `BillingOrder`（`pending`）。
3. **立即**标记 `paid` 并调用 `activateProFromPayment`。
4. 前端展示「当前为模拟支付」提示。

### 目标实现（商业计划可写进路线图）

1. 创建订单 → 跳转微信/支付宝。
2. 支付回调 → 更新订单 → 开通 Pro。
3. 发票与对公（ToB 扩展）。

### 订单状态

| status | 含义 |
|--------|------|
| pending | 待支付 |
| paid | 已支付 |
| failed | 失败 |
| refunded | 已退款 |

### 退款规则

- 用户可在账单页申请退款（已支付订单）。
- 退款后 `revokeProAfterRefund`，**Pro 权益立即收回**。
- 前端二次确认：「退款后 Pro 权益将立即收回」。

### 订阅生命周期

- `cancelAtPeriodEnd`：到期后不再续费，降回免费版。
- `resumeSubscription`：恢复自动续费。
- Pro 周期结束且未续费 → 自动降为免费版并重置额度周期。

## 单位经济模型（框架，需填入实际成本）

商业计划书可按下表测算（数字需运营填写）：

| 变量 | 符号 | 说明 |
|------|------|------|
| Pro 月 ARPU | P | ¥29（当前定价） |
| 月付占比 | — | 影响现金流 |
| 平均 LLM 成本/次 | C | MiniMax + DeepSeek 混合 |
| 月均使用次数/Pro 用户 | U | 产品数据 |
| 毛利率（订阅） | (P - U×C) / P | 核心健康度 |
| 免费用户获客成本 | CAC | 营销费用/新增注册 |
| 免费→Pro 转化率 | r | 决定 LTV |

**LTV 粗算**：`LTV ≈ P × 平均留存月数 × r`（需减去退款与渠道费）。

## 定价策略叙事

| 策略 | 依据 |
|------|------|
| 免费版 30 次 | 够完成 1–2 篇完整三步流程体验，形成习惯 |
| Pro ¥29/月 | 低于多数独立创作者工具心理价位带，对标「一杯咖啡」 |
| 年付 ¥288 | 提高 LTV、降低 churn |
