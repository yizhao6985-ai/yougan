# 08 · 路线图与里程碑素材

供 BP「运营计划 / 里程碑与资金用途」章节使用。

---

## 1. 当前成熟度（基线）【已核验】

| 模块 | 状态 |
|------|------|
| 三步创作 + Web 创作台 + 文生图 | 可演示、开发环境生产可用 |
| 账号、作品、公域发布、发现灵感 | 可演示 |
| 会员与订单逻辑 | Beta（模拟支付） |
| 平台 OAuth 一键发布 | 部分实现（需分平台配置） |
| 真实支付、内容审核、App 上架 | 建设中 |

## 2. 待建设能力（缺口清单）

- 真实支付通道（微信/支付宝）与对账、发票；
- 单平台（小红书）发布闭环与失败降级；
- 公开内容审核策略、举报/下架流程、用户协议与隐私政策；
- 手机 App 客户端与下载分发；
- AI 额度加购包、团队/MCN 版、模板市场；
- 创作者数据看板；
- 生成式 AI 备案与内容标识合规。

## 3. 产品里程碑（建议口径，时间待定）

| 时间 | 里程碑 | 验收标准 |
|------|--------|----------|
| M1–M2 | 真实支付上线 | 微信/支付宝可完成开通；订单可对账 |
| M2–M3 | 小红书发布闭环 | 授权→发布成功率 >90%（测试账号） |
| M3 | 种子计划完成 | 50–100 创作者；制作计划定稿率 >40% |
| M4–M6 | 发现页运营 + SEO | 发现页 UV【待填】；自然注册占比 >20% |
| M6 | App MVP | 记灵感 + 账号同步可用 |
| M6–M9 | 2–3 平台 OAuth | 公众号等上线 |
| M9–M12 | 团队版试点 | ≥3 家 MCN/工作室付费 |

## 4. 资金用途（示意 · 融资 400 万为例）【待填实际金额】

| 用途 | 占比 | 金额（万） | 说明 |
|------|------|------------|------|
| 研发 | 30% | 120 | 支付、平台发布、App、Pro 差异化 |
| 获客与运营 | 30% | 120 | 种子计划、内容、社群 |
| 基础设施与 AI 成本 | 20% | 80 | 云、LLM 预充值、安全合规 |
| 人力补充 | 15% | 60 | 运营/客服/商务 |
| 储备金 | 5% | 20 | 不可预见支出 |

## 5. 下一轮融资触发条件（建议）

- MRR ≥【待填，例：15 万】且连续 3 月环比 >15%；
- LTV/CAC > 2；
- 单平台发布的活跃贡献可量化；
- 团队版首单签约。
