## 2. 单位经济公式（与 `05` 一致）

```text
单用户月毛利 ≈ ARPU − 人均实际 API 成本 − 支付手续费 − 固定成本分摊
LTV         ≈ ARPU × 平均留存月数
免费用户贡献 ≈ 转化率 × LTV − CAC
LTV/CAC     ≥ 2（健康目标）
```

**毛利关键变量**：人均 microCredits 消耗。因额度=成本上限，需监控逼近上限的重度用户占比，并以加购包/默认模型降配/限速作为护栏。

## 4. 敏感性分析（应分析的变量）

| 变量恶化 | 影响 |
|----------|------|
| 转化率 −2pp | MRR 显著下降，跑道缩短 |
| 人均 API 成本 +30% | 毛利率压缩，需调价/降配/限速 |
| CAC +50% | LTV/CAC 可能 <1，需收缩投放 |
| 付费流失 +3pp | LTV 显著下降 |
