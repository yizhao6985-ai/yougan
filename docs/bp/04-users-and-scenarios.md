# 04 · 目标用户与场景素材

供 BP「目标市场与客户细分」章节使用。

---

## 1. 主要客户画像

**画像 A：个人自媒体创作者**

- 一人运营 1–3 个平台，周更或日更图文/笔记。
- 主力平台：小红书、微信公众号。
- 付费动机：省时间、少返工、稳定更新、文案配图一站式。

**画像 B：垂类内容运营者**

- 职场、知识、生活方式、母婴、测评等垂类。
- 需要固定受众语气与结构模板（对应「固定设定 + 结构大纲 + 规则」）。
- 付费动机：方案复用、更高额度、多作品并行。

**画像 C：高频更新的 Pro/Pro+ 用户**

- 月均数十次以上 AI 辅助创作，常含出图。
- 免费版预算额度不足以支撑日更，驱动升级到 Pro / Pro+。

## 2. 次级用户

| 用户 | 行为 | 对业务价值 |
|------|------|-----------|
| 内容学习者 | 浏览发现灵感找写法 | 获客、SEO 流量 |
| 品牌型创作者 | 经营个人主页与公开作品集 | 留存、口碑传播 |

## 3. 典型使用场景【已核验：均可在产品中走通】

**场景 A：从零写一篇小红书种草**
定方案确认平台与选题 → 制作子图编排计划 → 出稿并配图（文生图）→ 发布到有感或小红书。

**场景 B：有方向、结构不清**
用户说「开始制作」，production 子图按方案自动编排任务 → 执行 → 产出 preview。

**场景 C：成稿后改语气/标题**
production 子图将修改记入待执行项 → 统一执行 → 生成执行摘要便于版本对比。

**场景 D：图文配图一体**
设计任务按方案画幅出图，物化到自有存储，前端图册展示，随作品一起发布。

**场景 E：多端协作（规划）**
手机记灵感 → Web 同一作品续写与发布。

## 4. 非目标用户（当前版本）

- 专业长视频剪辑全流程替代；
- 企业多层级审批与多席位权限（留待团队版）；
- 仅需排版、不需要 AI 的纯工具用户。
