# 11 · 术语表

供 BP 附录使用。撰写对外稿时，建议把内部字段名改写为投资人可读表述。

---

## 产品与业务术语

| 术语 | 含义 |
|------|------|
| 作品（Work） | 一次完整创作的容器，对应一段持续 AI 对话与一个 LangGraph thread |
| 定方案（profile） | 整理主题、体裁、表达、结构与规则的环节；不直接产出作品 |
| 备参考（reference） | 上传并分析参考素材（文/图/音/视频）的环节 |
| 制作（production） | AI 团队排计划并按方案执行出稿与配图的环节 |
| 提问（ask） | 优化、学习、背景答疑的环节；不擅改作品与计划 |
| 作品面板 | 创作台右侧四 Tab：方案 / 参考 / 作品 / 版本 |
| 作品预览（preview） | 制作产出的标题、正文与配图 |
| 发现灵感 | 公域内容浏览页（`/content`），用于选题参考与获客 |
| 制作计划定稿 | 用户确认制作计划完成，可继续出稿 |
| 创意度 | 调节制作环节 AI 发散程度的滑杆（0.1–1.0），不影响定方案/提问/计划 |

## 商业与计量术语

| 术语 | 含义 |
|------|------|
| AI 创作额度 | 用户侧对「本周期可消耗 API 成本预算」的称呼，按百分比展示 |
| microCredit（mc） | 内部计量单位，`1 mc = ¥0.0001`，`¥1 = 10,000 mc`，对齐 LLM 官方成本 |
| 月度预算额度 | 套餐每周期可消耗的 microCredits 上限（成本上限，非固定次数） |
| 套餐（plan） | free / pro / pro_plus 三档 |
| ARPU | 每付费用户平均收入 |
| LTV | 用户生命周期价值 ≈ ARPU × 平均留存月数 |
| CAC | 单用户获客成本 |
| MRR / MAU | 月度经常性收入 / 月活跃用户 |

## 技术术语

| 术语 | 含义 |
|------|------|
| 回合队列（turn queue） | 每条用户消息自动路由到对应环节子图的机制 |
| LangGraph | Agent 工作流框架；会话状态以 checkpoint 持久化 |
| thread / checkpoint | 一段对话的运行时标识与持久化快照 |
| 物化（materialize） | 把 Agent 产出的临时外链图片转存到自有存储（OSS/local） |
| 版本时间轴 | 作品产出后记录的单线版本历史，可回滚 |
| OpenAPI | 前后端契约规范，前端类型由其生成 |
| OSS | 阿里云对象存储，图片与静态资源的生产存储驱动 |

> 完整产品/技术名词另见 `docs/business/glossary.md`。
