## 3. 计量与计费机制【已核验 · 与旧文档不同】

- 计量单位 **microCredits**：`1 mc = ¥0.0001`，按 **AI 实际算力成本**计（与百炼/DeepSeek/MiniMax 官方价对齐）。
- 每次 LLM 调用按 token 用量换算 microCredits，文生图按 ¥0.4/张；缓存命中走缓存价。
- 月度额度是**可消耗的成本预算上限**，非固定「次数」；用户侧展示为「本月 AI 创作额度」百分比。
- 超额：提示升级或下月按周期重置；订单/退款记录于 `BillingOrder`。

> 这是从「计次」转向「计成本」的模型，定价与底层 LLM 成本天然对标，抗模型涨价/降价冲击能力更强。

## 4. 定价逻辑

- **免费预算 ¥80**：约够 1–2 篇含配图的完整流程，形成使用习惯。
- **Pro ¥68 / Pro+ ¥128**：覆盖周更与日更两档需求，价格仍在创作者工具心理账户内。
- **年付约 8.3 折**：提升 LTV、降低流失。
- **平台发布绑定付费档**：强化付费理由，区别于纯聊天 AI。

## 5. 单位经济模型【框架 + 假设，须实测替换】

> 关键提醒：额度是**成本上限**而非预期用量。单位经济必须按**真实人均消耗**建模，而非按预算上限。

**变量定义**

| 符号 | 含义 | 初稿假设（待实测） |
|------|------|--------------------|
| P | Pro 月 ARPU（混合年付） | ¥60 |
| Cᵤ | Pro 用户月均实际 API 成本 | 【待实测，远低于 ¥255 上限，例 ¥15–40】 |
| r | 免费 → 付费转化率 | 5%（种子期假设） |
| L | 付费平均留存月数 | 8 个月 |
| CAC | 单注册获客成本 | ¥30（目标，待验证） |

**单用户毛利（Pro，月）**

```text
毛利 ≈ P − Cᵤ − 支付手续费 − 固定成本分摊
例：60 − 25 − 通道费 ≈ 正毛利（取决于 Cᵤ 实测）
```

**LTV（示意）**

```text
LTV ≈ P × L = 60 × 8 = ¥480（未扣退款与手续费）
```

**免费用户贡献**

```text
贡献 ≈ r × LTV − CAC
例：5% × 480 − 30 ≈ −¥6/注册（早期 CAC 偏高时可能为负，靠留存与年付改善）
```

**毛利护栏**：因额度=成本上限，需监控「重度用户消耗逼近上限」的占比；必要时通过加购包、降配默认模型或限速保护毛利。

## 6. 支付与订阅运营【已核验】

- **目标流程**：下单 → 微信/支付宝 → 回调开通 → 账单与发票。
- **当前**：模拟支付，仅用于产品与转化漏斗测试（文案已标明）。
- **退款**：支持订单退款，付费权益立即收回（降低付费顾虑）。
- **续费**：支持到期取消续费（`cancelAtPeriodEnd`）；周期结束未续费自动降回免费版。
