## 2. 投资人常见问答（Q&A 素材）

**Q：和 ChatGPT/豆包有何不同？**
A：通用 AI 一问一文、过程不可控；有感把创作拆成可确认的「定方案→制作→提问」流水线，确认后再出稿与配图，并连到发布。产品边界由代码保证（定方案不出稿、制作不跳过任务、提问不擅改作品）。

**Q：你们的护城河？**
A：流程与高转化方案数据、发现页 + 作者主页的生态、OAuth 分发绑定提高迁移成本、垂类深耕、按成本计量的成本工程。详见 `06-competition.md`。

**Q：怎么赚钱？毛利如何？**
A：三档订阅（免费/¥68/¥128），按 microCredits（API 实际成本）计量额度。毛利取决于人均真实消耗远低于额度上限；须用上线后运营数据建模（`05`、`09`）。

**Q：额度是成本上限，会不会被薅穿？**
A：额度为周期成本预算上限，且有计量结算与超额提示；设加购包、默认模型降配与限速作为护栏，并监控重度用户占比。

**Q：现在能演示什么？真实支付呢？**
A：核心创作、文生图、公域发布、会员与额度均可演示；当前为模拟支付，真实支付通道为近期里程碑（`08`）。

**Q：平台政策风险怎么办？**
A：分平台分批上线、失败降级；始终保留「发布到有感」公域出口；推进与平台官方合作。

## 3. 尽职调查演示清单（Due Diligence）

1. 注册登录 → 新建作品 → 开屏建议；
2. 定方案确认 → 开始制作 → 产出 preview（含文生图配图）；
3. 创意度调整 → 再次执行 → 执行摘要与版本对比；
4. 发布到有感 → 发现页可见 → 个人主页统计；
5. 会员额度展示 → 套餐升级（测试环境）→ 平台集成页。
