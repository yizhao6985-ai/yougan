# 07 · 市场进入与增长素材

供 BP「市场进入与增长策略 / 运营」章节使用。按 AARRR 漏斗组织。

---

## 1. 获客（Acquisition）

| 渠道 | 动作 | KPI |
|------|------|-----|
| 落地页 + 能力页 | SEO，「AI 自媒体」「小红书选题」等长尾词 | 注册转化率 |
| 发现灵感公域 | 精选内容、分享 slug、作者主页外链 | 访客 → 注册 |
| 种子创作者计划 | 邀请码、案例访谈、社群答疑 | 有效注册数 |
| 内容营销 | 三步 SOP 教程、对比「一问一文」 | 品牌搜索量 |
| 合作 | 自媒体训练营、MCN 工具采购试点 | B 端线索 |

## 2. 激活（Activation）

**北极星激活事件**：注册后 24 小时内「创建 1 件作品 + 完成 1 次定方案对话」。

| 手段 | 说明（产品已具备） |
|------|--------------------|
| 开屏建议 | 空 thread 时给 7 条 AI 下一步建议，降低冷启动 |
| 空状态引导 | 提示「定方案 → 制作」路径 |
| 邮件/站内召回 | 未完成首作品用户（规划中） |

## 3. 转化（Revenue）

| 触发点 | 机制 |
|--------|------|
| 额度将尽/用尽 | 会员页提示「已使用本月额度 x%」 |
| 制作计划定稿后 | 体验完整价值后再触达付费 |
| Pro 功能预览 | 平台集成页展示绑定价值 |
| 年付折扣 | 约 8.3 折降低价格敏感 |
| Pro → Pro+ | 重度用户额度不足时升档 |

**目标转化率（种子期假设，待验证）**：注册 → 付费 **3–8%**。

## 4. 留存（Retention）

| 机制 | 说明（产品已具备） |
|------|--------------------|
| 作品云同步 | 换设备续写 |
| 作品面板 + 版本时间轴 | 切换作品即可回溯决策、回到任意版本 |
| 分组管理 | 多栏目/系列并行 |
| 平台账号绑定 | 提高迁移成本 |
| 发现页曝光 | 发布获得阅读反馈，正向激励 |

## 5. 自传播（Referral）

- 发现页公开作品 + 作者主页 → 外部分享带来访客 → 注册。
- 创作飞轮：创作 → 发布到有感 → 公域曝光 → 访客注册 → 成为创作者。

## 6. 核心 KPI 体系【框架，数值待运营填充】

| 阶段 | 指标 |
|------|------|
| 获客 | UV、注册率、CAC |
| 激活 | 首作品率、首对话率、D1 留存 |
| 参与 | 制作计划定稿率、月活跃作品数、人均 microCredits 消耗 |
| 变现 | 免费→付费转化率、MRR、ARPU、退款率、Pro+ 占比 |
| 留存 | M3 留存、付费 churn、年付占比 |
| 生态 | 公开发布率、发现页 PV/UV、篇均阅读 |

## 7. 运营节奏

### 平台接入优先级

| 优先级 | 平台 | 理由 |
|--------|------|------|
| P0 | 小红书 | 与产品文案、种子用户画像一致 |
| P1 | 微信公众号 | 图文主力场景 |
| P2 | 微博、抖音 | 扩展覆盖 |
| P3 | 快手、B 站、海外（Twitter/LinkedIn/Instagram） | 补齐平台矩阵 |

每平台需：开放平台应用、OAuth 环境变量、回调 URL、发布 API 联调、失败降级与客服话术。

### 公域「发现灵感」运营

- **冷启动**：官方/种子账号发布高质量案例，建立筛选维度样本。
- **精选位**：首页「精选推荐」人工 + 规则。
- **话题活动**：带动 UGC（如 #选题挑战）。
- **合规**：上线前完成公开内容审核策略与用户协议（当前缺口，P2 必建）。
