# 快速上手有感

## 注册与登录

1. 打开有感网站，点击「开始创作」或「登录」。
2. **注册**：使用邮箱 + 密码（至少 6 位），可选填昵称。
3. **登录**：使用注册邮箱登录，作品与制作记录会**云端保存**，换设备可继续。

忘记密码可在「忘记密码」页申请邮件重置（需配置邮件服务；本地开发时链接可能出现在控制台）。

## 推荐：第一件作品怎么写

### 第一步：新建作品

1. 进入创作台（`/studio`）。
2. 点击「新建作品」，输入作品标题（如「小红书护肤种草」）。
3. 一件作品 = 一段持续对话 + 独立的制作状态，可按系列分组管理。

### 第二步：定制作方案

空作品打开后，系统可能提供 **7 条开场建议**，点选即可发送，也可以直接输入，例如：

- 「我想写一篇小红书护肤种草，面向 25–35 岁职场女性」
- 「还没想好选题，帮我 brainstorm 三个方向」
- 「想做职场干货，不确定发小红书还是公众号」

对话中确认的内容会写入右侧作品面板的 **「方案」** 标签页。

**定方案阶段不会直接出正文**，目的是先把方向定稳。

### 第三步：开始制作

方案大致就绪后，在对话中说：

- 「开始制作」
- 「可以出稿了」
- 「按现在的方案写正文」

系统会进入制作环节：AI 团队先制定内部制作计划，再按计划产出标题、正文等内容，显示在 **「作品」** 标签页。

### 第四步：修改与提问

成稿后如需调整，直接说：

- 「标题更吸引人一点」
- 「语气更口语」
- 「第二段缩短」

制作过程中也可以随时提问，例如「小红书标题有什么写法技巧？」

### 第五步：发布（可选）

满意后可在创作台 **「发布到有感」**，作品会出现在「发现灵感」公域；Pro 会员还可绑定第三方平台发布。

## 完整推荐流程一览

```
新建作品 → 定方案（对话确认主题/平台/结构）
         → 备参考（可选，上传对标素材）
         → 说「开始制作」→ 查看「作品」面板
         → 修改 / 提问 → 发布
```

## 开屏建议怎么用

- 新建作品且尚无对话时，系统生成可点击的**下一步建议**。
- 每条建议对应一种常见创作起点，点选即发送，省去自己想第一句话的时间。
- 建议不适用时，直接在输入框描述你的情况即可。

## 额度说明（简要）

每次在创作台与 AI 成功对话会消耗本月 **AI 创作额度**（以使用百分比展示）。免费版可体验完整流程；高频使用可升级 Pro / Pro+。详见 [10-membership-and-billing.md](./10-membership-and-billing.md)。

## 常见新手误区

| 误区 | 正确做法 |
|------|----------|
| 希望第一句就出全文 | 先花几轮对话定方案，再说「开始制作」 |
| 换平台或换选题仍用同一件作品 | 另存为新作品，避免方案与成稿混杂 |
| 只在聊天里看结果 | 养成看右侧「方案」「作品」「版本」面板的习惯 |
| 忽略开场建议 | 建议可快速进入状态，也可跳过 |
