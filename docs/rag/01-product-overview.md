# 有感是什么

## 一句话介绍

**有感（Yougan）** 是一款 **AI 创作助手**，帮你先想清楚要做什么，再按计划做出文章、笔记、脚本、插画等作品，全程可以确认、修改和回溯。

## 有感解决什么问题

自媒体和内容创作者常遇到三类困扰：

1. **没选题**：不知道写什么、发哪个平台、面向谁。
2. **方向不稳**：很多 AI 工具「问一句出一篇」，选题偏了就要整篇重写，浪费时间和额度。
3. **发布链路长**：在多个工具里写完，还要复制到小红书、公众号等平台。

有感把创作拆成**可确认、可回溯**的步骤，让 AI 在每一步只做该做的事，你在关键节点拍板。

## 有感和其他 AI 工具有什么不同

| 常见做法 | 有感的做法 |
|----------|------------|
| 一次 Prompt 直接出全文 | 先定制作方案，再排计划、再出稿 |
| 只有聊天记录 | 右侧「作品面板」沉淀方案、参考、作品、版本 |
| 选题错了全文重来 | 在方案阶段就能纠偏，减少返工 |
| 写完自己复制发布 | 可发布到「发现灵感」，Pro 可绑定平台一键发布 |

有感适合**需要持续更新内容**的创作者，而不是偶尔问 AI 一个问题。

## 核心创作流程

有感推荐按以下环节推进一件作品：

1. **定方案** — 确认主题、体裁、受众、语气、内容结构
2. **备参考**（可选）— 上传对标内容或素材，记录借鉴意图
3. **制作** — AI 团队制定计划并按步骤产出作品
4. **提问**（随时）— 咨询优化方向、创作方法，不直接代替制作

你说「开始制作」后，系统会进入制作环节；制作过程中也可以继续提问或调整方案。

## 支持的创作形态

有感可辅助多种内容形态，例如：

- 观点长文、清单笔记、案例故事
- 教程干货、对比评测
- 脚本口播、短视频脚本
- 插画绘画（按方案执行）

具体产出取决于你在方案阶段确认的体裁与媒介。

## 产品入口

| 入口 | 用途 |
|------|------|
| Web 创作台（`/studio`） | 主要创作场景：对话 + 作品面板 |
| 发现灵感（`/content`） | 浏览他人公开作品，找选题参考 |
| 产品能力页（`/features`） | 了解各环节能做什么、不能做什么 |
| 手机 App | 记录灵感，与 Web 同一账号同步（下载链接受部署配置） |

## 有感目前不擅长什么

以下场景可能不太适合只用有感：

- 专业视频剪辑与重度后期（抖音/B 站以内容策划和脚本为主）
- 企业多账号审批与协作（当前为个人账号模型）
- 完全不需要 AI、只要排版的纯工具需求
