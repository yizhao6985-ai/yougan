# 如何制作与修改作品

## 如何进入制作环节

在方案大致就绪后，在对话中明确表达出稿意图，例如：

- 「开始制作」
- 「按现在的方案出稿」
- 「可以写了」
- 「帮我生成标题和正文」

系统识别到**明确出稿意图**后，会进入制作流程，而不是继续在方案里讨论。

> 若你只是在描述方向、讨论选题，尚未要求出稿，系统会留在定方案环节——这是为了减少「方向还没定就写出全文」的返工。

## 制作环节会发生什么

制作由 AI 团队在内部分步完成，你通常**不需要逐步确认每个子任务**：

1. **制定制作计划** — 根据当前方案编排任务（如写标题、写正文、写标签等）
2. **按任务执行** — 文字、视觉等按专长分工完成
3. **整合成稿** — 产出写入作品面板 **「作品」** 标签页
4. **记录版本** — 每次产出会新增一版，可在 **「版本」** 中查看

对话区可能显示「AI 团队正在执行制作」等进度提示。

## 支持的产出类型

取决于方案中的体裁与媒介，常见包括：

- 标题、正文、话题标签
- 分镜/口播脚本
- 插画、封面方向描述
- 清单体、教程步骤

当前版本以**图文与文案**为主；音视频成品能力在持续扩展中。

## 如何修改成稿

成稿后直接在对话中说明修改意见，例如：

| 修改类型 | 示例说法 |
|----------|----------|
| 标题 | 「标题更吸引人，但不要标题党」 |
| 语气 | 「全文语气再口语一点」 |
| 结构 | 「把第二段和第三段合并，加一个小标题」 |
| 长度 | 「压缩到 600 字以内」 |
| 细节 | 「产品 A 的描述改成强调性价比」 |

系统会把修改纳入制作流程并更新「作品」面板；新版本会出现在版本时间轴。

## 版本管理

### 版本时间轴

- 每次产出新的作品内容会记录一版
- 可查看历史版本对比
- **回到这一版**：恢复到某一历史版本继续编辑（当前进度会被替换，但历史记录保留）

### 另存为新作品

想换主题或换方向、但想保留当前进度作参考时：

- 使用 **「另存为新作品」**
- 从当前版本复制为新作品，原作品不受影响

## 创意度调节

创作台的 **创意度** 滑杆只影响制作环节 AI 的发散程度：

- **调低**：更忠实方案、更稳、适合事实类、教程类
- **调高**：更有创意、适合故事、种草、需要新鲜表达的内容

不影响定方案、提问和制作计划本身。

## 制作环节 AI 不会做什么

- 不会跳过制作计划直接「偷偷」生成（保证可追溯）
- 不会在提问环节直接改作品（需通过制作或明确改稿意图）

## 方案与制作的关系

| 情况 | 建议 |
|------|------|
| 成稿整体方向错了 | 先回对话调整方案，再重新制作 |
| 只是局部措辞、语气问题 | 直接说修改意见即可 |
| 想换平台或换选题 | 另存为新作品，重新定方案 |

## 制作完成的下一步

1. 在「作品」面板通读成稿
2. 必要时继续修改或回到历史版本
3. 满意后 **发布到有感** 或（Pro）发布到绑定平台
