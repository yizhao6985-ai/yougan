# 发布作品与发现灵感

## 发布到有感

### 前提

创作台「作品」面板已有成稿内容（完成制作环节）。

### 步骤

1. 在创作台点击 **「发布到有感」**
2. 打开发布确认弹窗 — 系统根据作品内容 **AI 推断**分类标签
3. 确认或修改以下标签：
   - **内容体裁**（如图文笔记、长文深度）
   - **主题类别**（如生活方式、知识干货）
   - **媒介形态**（纯文字、图文、音频等）
4. 确认发布

### 为什么发布时要确认分类

创作过程中体裁可能变化；**发布是对外承诺的时刻**，此时确认标签成本最低，也便于他人在发现页找到你的内容。

### 发布后

- 作品进入 **已发布** 状态
- 创作台可出现「在发现灵感查看」链接
- 内容详情页 URL 形如 `/content/:slug`
- 阅读量会计入你的主页统计

## 管理已发布内容

路径：**设置 → 我的发布**（`/settings/publications`）

| 状态 | 含义 |
|------|------|
| 草稿 | 已保存未公开 |
| 已发布 | 在发现页与主页展示 |
| 已归档 | 下线公开展示 |

## 发现灵感（公域浏览）

路径：`/content`

### 可以做什么

- 按意图快捷入口浏览：读故事、看干货、刷笔记、听内容、看视频
- 用筛选组合：体裁、主题、媒介
- 阅读他人公开作品，找选题与写法参考

### 对创作者的价值

- 降低选题枯竭 — 看同类型作品结构
- 发布后获得公域曝光与阅读数据
- 访客可能因此注册成为新创作者（生态飞轮）

## 个人主页

### 自己的主页 `/profile`

- 编辑头像、封面、简介
- 查看已发布列表与阅读趋势

### 公开主页 `/user/:userId`

- 他人可见你的已发布内容与统计
- **已发布篇数**、**总阅读量**、**近 6 月发布趋势**

可把公开主页当作对外 **作品集** 链接。

## 发布与会员

- **免费版** 可发布到有感、浏览发现页
- **Pro / Pro+** 提供更高 AI 创作额度与旗舰出稿能力
- 公域发布本身不单独收费

## 合规提示

发布公开内容前请确保：

- 不侵犯他人版权
- 遵守内容规范与广告披露要求
- 专业领域（医疗、金融等）内容需自行把关准确性

有感侧内容审核与投诉流程仍在完善中，作者对发布内容负责。
