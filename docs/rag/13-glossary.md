# 用户术语表

## 产品与账户

| 术语 | 解释 |
|------|------|
| **有感 / Yougan** | 本产品，AI 创作助手 |
| **创作台** | 登录后的主工作区（`/studio`），左侧作品、中间对话、右侧作品面板 |
| **作品** | 一次完整创作项目，包含方案、参考、成稿、版本与对话记录 |
| **作品分组** | 把多件作品归入同一栏目或系列 |
| **发现灵感** | 公域内容浏览页（`/content`），可看他人的已发布作品 |

## 创作环节

| 术语 | 解释 |
|------|------|
| **定方案** | 创作第一步：确认主题、体裁、受众、结构、规则，不直接出正文 |
| **备参考** | 上传并分析对标素材，记录借鉴意图（可选） |
| **制作** | AI 团队按计划产出标题、正文等成稿内容 |
| **提问** | 创作过程中的答疑与咨询，可问方法、优化建议 |
| **智能回合** | 每条消息自动识别应更新方案、执行制作还是答疑 |

## 作品面板

| 术语 | 解释 |
|------|------|
| **作品面板** | 创作台右侧面板，含方案、参考、作品、版本四个标签 |
| **制作方案** | 「方案」标签中的内容：主题、表达、结构、规则等 |
| **参考素材** | 「参考」标签中的对标内容与分析 |
| **作品内容** | 「作品」标签中的成稿（标题、正文等） |
| **版本记录** | 「版本」标签中的历史成稿时间轴 |

## 方案字段（用户可见）

| 术语 | 解释 |
|------|------|
| **制作规格** | 主题、体裁、媒介形式 |
| **表达设定** | 受众、语气、视觉风格 |
| **内容结构** | 分段节拍（开头、章节、场景等顺序） |
| **制作规则** | 必须遵守或避免的约束 |
| **体裁参数** | 字数、画幅、时长等量化要求 |
| **创意度** | 滑杆，调节制作时 AI 发散程度 |

## 制作与版本

| 术语 | 解释 |
|------|------|
| **制作计划** | 系统内部编排的制作任务列表，用户一般通过对话间接触发 |
| **AI 团队** | 产品中对文字、视觉等分工执行制作的拟人化说法 |
| **回到这一版** | 恢复到某一历史版本继续编辑 |
| **另存为新作品** | 从当前进度复制出新作品，不影响原作 |

## 发布与社区

| 术语 | 解释 |
|------|------|
| **发布到有感** | 将成稿公开发布到发现灵感与个人主页 |
| **发布确认** | 发布前确认或修改 AI 推断的内容分类标签 |
| **内容体裁** | 如图文笔记、长文深度、短帖动态 |
| **主题类别** | 如生活方式、知识干货、故事叙事 |
| **媒介形态** | 纯文字、图文、音频、视频等 |
| **个人主页** | 展示已发布作品与阅读数据的作者页 |

## 会员与计费

| 术语 | 解释 |
|------|------|
| **AI 创作额度** | 每月可用于 AI 对话的用量上限，以使用百分比展示 |
| **免费版 / Pro / Pro+** | 三档会员套餐 |
| **平台一键发布** | Pro 能力：绑定 OAuth 后直接发到第三方平台 |

## 操作方式

| 术语 | 解释 |
|------|------|
| **侧栏直改** | 在作品面板直接编辑字段，保存即生效，无 AI 回复 |
| **开场建议** | 新作品时系统生成的可点击第一句建议 |
| **下一步建议** | 每轮对话结束后给出的可点击后续操作建议 |
