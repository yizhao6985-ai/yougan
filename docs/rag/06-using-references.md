# 如何使用参考素材

## 参考素材有什么用

参考素材帮你把「心里想对标的样子」告诉 AI，例如：

- 一篇爆款笔记的结构和节奏
- 某种语气或排版风格
- 产品图片、封面视觉
- 口播视频的叙事方式

有感会分析素材内容，并结合你声明的**借鉴意图**，在定方案和制作时作为约束，而不是简单洗稿。

## 如何添加参考

### 方式一：对话附件

在发送消息时添加附件：

- **图片** — 封面、截图、产品图
- **音频** — 播客片段、口播样例
- **视频** — 短视频参考
- **文本** — 粘贴的对标文案

单次消息最多 **6 份**参考素材。

发送后系统会自动分析新附件，分析结果汇总到作品面板 **「参考」** 标签页。

### 方式二：侧栏管理

在「参考」面板可查看、展开每条素材的：

- **分析** — AI 对内容结构、风格等的摘要
- **借鉴意图** — 你或 AI 记录的「要学什么」

也可在侧栏直接删改参考条目（会同步到对话上下文）。

## 如何说清楚「借鉴什么」

上传素材时或之后，建议说明借鉴范围，例如：

| 好的表述 | 说明 |
|----------|------|
| 「学它的清单结构和开头钩子，不抄具体内容」 | 限定借鉴维度 |
| 「参考语气，话题换成我的产品」 | 明确替换关系 |
| 「学封面配色和排版，文字我自己写」 | 区分视觉与文案 |

避免只说「照着这个写」——容易让成稿过于雷同，也不利于平台原创要求。

## 参考 vs 方案 vs 制作

| 环节 | 与参考的关系 |
|------|--------------|
| 定方案 | 参考可影响结构、语气等方案字段 |
| 制作 | 按计划产出，参考作为约束输入 |
| 提问 | 可问「这篇参考好在哪里」，但不直接改参考库 |

参考素材**不会自动替代**你的方案；你仍需确认主题、受众和规则。

## 使用参考的最佳实践

1. **精选少量高质量对标** — 1–3 篇往往比堆 10 篇更有效
2. **写明借鉴意图** — 结构、语气、选题角度、视觉，选一两项即可
3. **用自己的信息和观点** — 参考是「写法」，内容应是你的真实经历或调研
4. **注意版权与平台规则** — 不要上传无授权的完整他人作品用于抄袭

## 常见问题

**上传后没反应？**  
分析需要一些时间，对话区可能显示「正在分析参考素材」。完成后在「参考」面板查看。

**参考和方案冲突怎么办？**  
在对话中说明以哪边为准，例如「结构按参考 A，语气按方案里定的口语风格」。

**可以不要参考直接做吗？**  
可以。参考是可选环节，方案足够清晰时可直接「开始制作」。
