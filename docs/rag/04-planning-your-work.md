# 如何定好制作方案

## 定方案是做什么的

**定方案**是有感创作的第一步，目标是回答：

- 写什么（主题、角度）
- 什么形式（笔记、长文、脚本等）
- 写给谁（受众）
- 怎么写（语气、结构、规则）

定方案**不会直接产出正文**。方案确认后，再说「开始制作」进入出稿环节。

## 建议在方案阶段确认的信息

### 1. 主题与角度

越具体越好。对比：

| 较模糊 | 较清晰 |
|--------|--------|
| 写一篇护肤 | 秋冬干皮通勤族的 3 款平价面霜测评 |
| 职场内容 | 工作 3 年的人如何向上沟通要资源 |

### 2. 体裁与媒介

说明作品是什么形式，例如：

- 图文笔记、观点长文、清单体短帖
- 短视频脚本、播客提纲、插画系列

AI 会按体裁整理篇幅、结构与表达方向。

### 3. 受众与语气

示例表述：

- 「面向 25–35 岁一线城市职场女性，语气亲切但不幼稚」
- 「面向编程初学者，语气专业、少黑话」
- 「面向品牌方市场负责人，语气简洁有数据感」

### 4. 内容结构

可以说你希望的结构，例如：

- 笔记：痛点钩子 → 3 个要点 → 总结避坑
- 长文：故事开头 → 问题分析 → 方法 1/2/3 → 行动号召
- 清单体：5 条可执行建议，每条带小标题

确认后会显示在方案面板的 **「④ 结构与要素」** 中。

### 5. 制作规则（可选但推荐）

明确边界能减少返工，例如：

- 「不出现竞品具体价格」
- 「全文不超过 800 字」
- 「必须包含个人真实使用体验，不编造」
- 「标题不要用震惊体」

## 对话示例

**从零开始：**

> 我想做居家收纳方向的内容，但还没想好具体选题，帮我给 3 个适合新手的角度。

**已有方向，补全方案：**

> 选题定了：租房党小厨房收纳。体裁是图文笔记，受众 22–30 岁租房女生，语气实用亲切。帮我定一个结构。

**调整方案：**

> 语气太正式了，改成更像朋友分享的口语。结构里加一段「踩坑对比」。

**准备进入制作：**

> 方案可以了，开始制作吧。

## 侧栏「方案」面板（五步向导）

| 步骤 | 含义 |
|------|------|
| ① 创作定位 | 这篇作品要做什么 |
| ② 体裁与参数 | 体裁、媒介与字数/画幅/时长等 |
| ③ 表达设定 | 受众、语气、视觉/氛围方向 |
| ④ 结构与要素 | 固定设定 + 结构大纲（段落、分镜、章节等） |
| ⑤ 创作规则 | 必须遵守或需要避免的约束 |

对话确认后自动填入对应步骤；也可在侧栏直接编辑。**必填就绪**：① 创作定位 + ② 体裁。

## 定方案环节 AI 不会做什么

- 不会直接写出完整正文
- 不会执行制作任务或跳过方案出稿

若你只想问「标题怎么写更吸引人」，属于**提问答疑**，AI 会给建议；若你说「把标题风格记入方案」，系统会整理进制作方案。

## 何时算「方案就绪」

满足以下即可说「开始制作」：

- [ ] **创作定位**清楚（一句话说清要做什么）
- [ ] **体裁**已选（如图文笔记、短视频脚本、插画系列等）
- [ ] （建议）受众与语气有共识
- [ ] （可选）结构大纲与创作规则

不必追求完美；制作环节仍可修改，但方案越清晰，返工越少。

## 换选题怎么办

不要在同一件作品里硬换完全不同的方向。建议：

1. **另存为新作品**，或
2. **新建作品**重新定方案

这样方案、参考和版本不会互相污染。
