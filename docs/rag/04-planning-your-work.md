# 如何定好制作方案

## 定方案是做什么的

**定方案**是有感创作的第一步，目标是回答：

- 写什么（主题、角度）
- 什么形式（笔记、长文、脚本等）
- 发给谁（平台、受众）
- 怎么写（语气、结构、规则）

定方案**不会直接产出正文**。方案确认后，再说「开始制作」进入出稿环节。

## 建议在方案阶段确认的信息

### 1. 主题与角度

越具体越好。对比：

| 较模糊 | 较清晰 |
|--------|--------|
| 写一篇护肤 | 秋冬干皮通勤族的 3 款平价面霜测评 |
| 职场内容 | 工作 3 年的人如何向上沟通要资源 |

### 2. 目标平台

不同平台决定篇幅、语气和结构。有感支持面向：

- 小红书、微博、微信公众号
- 抖音、快手、哔哩哔哩
- 有感「发现灵感」

在对话中说明「主要发小红书」或「公众号长文」，AI 会按平台习惯整理方案。

### 3. 受众与语气

示例表述：

- 「面向 25–35 岁一线城市职场女性，语气亲切但不幼稚」
- 「面向编程初学者，语气专业、少黑话」
- 「面向品牌方市场负责人，语气简洁有数据感」

### 4. 内容结构

可以说你希望的结构，例如：

- 小红书笔记：痛点钩子 → 3 个产品 → 总结避坑
- 公众号长文：故事开头 → 问题分析 → 方法 1/2/3 → 行动号召
- 清单体：5 条可执行建议，每条带小标题

确认后会显示在方案面板的 **「内容结构」** 中。

### 5. 制作规则（可选但推荐）

明确边界能减少返工，例如：

- 「不出现竞品具体价格」
- 「全文不超过 800 字」
- 「必须包含个人真实使用体验，不编造」
- 「标题不要用震惊体」

## 对话示例

**从零开始：**

> 我想做小红书内容，方向是居家收纳，但还没想好具体选题，帮我给 3 个适合新手的角度。

**已有方向，补全方案：**

> 选题定了：租房党小厨房收纳。平台小红书，受众 22–30 岁租房女生，语气实用亲切。帮我定一个笔记结构。

**调整方案：**

> 语气太正式了，改成更像朋友分享的口语。结构里加一段「踩坑对比」。

**准备进入制作：**

> 方案可以了，开始制作吧。

## 侧栏「方案」面板各字段

| 字段 | 含义 |
|------|------|
| 制作规格 | 主题、体裁、媒介（图文/纯文字等） |
| 表达设定 | 受众、语气、视觉风格 |
| 内容定位 | 这篇内容的核心卖点或立场 |
| 创作设定 | 固定人物、背景、世界观等（故事类常用） |
| 内容结构 | 分段节拍与顺序 |
| 制作规则 | 必须遵守的约束 |
| 体裁参数 | 字数、画幅、时长等量化要求 |

对话确认后自动填入；也可在侧栏直接编辑。

## 定方案环节 AI 不会做什么

- 不会直接写出完整正文
- 不会执行制作任务或跳过方案出稿

若你只想问「小红书标题怎么写」，属于**提问答疑**，AI 会给建议；若你说「把标题风格记入方案」，系统会整理进制作方案。

## 何时算「方案就绪」

满足以下即可说「开始制作」：

- [ ] 主题和角度清楚
- [ ] 目标平台明确（或默认有感）
- [ ] 受众和语气有共识
- [ ] 有基本内容结构（或信任 AI 按平台惯例编排）

不必追求完美；制作环节仍可修改，但方案越清晰，返工越少。

## 换选题或换平台怎么办

不要在同一件作品里硬换完全不同的方向。建议：

1. **另存为新作品**，或
2. **新建作品**重新定方案

这样方案、参考和版本不会互相污染。
