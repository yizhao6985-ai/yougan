# 账号、设置与平台绑定

## 设置入口

登录后从创作台或导航进入 **个人设置**，主要分区：

| 分区 | 包含页面 |
|------|----------|
| 账户 | 资料与外观、账号与安全 |
| 会员与计费 | 会员、订单与支付 |
| 创作与发布 | 作品管理、我的发布、平台集成 |
| 帮助 | 问题反馈 |

## 资料与外观

路径：`/settings/profile`

可设置：

- **昵称、简介** — 展示在个人主页
- **头像** — JPEG/PNG/WebP/GIF，不超过 3MB
- **主页封面** — 建议宽图，不超过 8MB

## 账号与安全

路径：`/settings/account`

- **修改登录邮箱** — 向新邮箱发确认链接，需输入当前密码；确认前仍用原邮箱登录
- **密码** — 通过忘记密码流程重置

## 作品管理

路径：`/settings/works`

集中查看和管理作品（与创作台左侧列表能力互补，适合批量整理）。

## 我的发布

路径：`/settings/publications`

管理发布到有感的内容：草稿、已发布、归档。浏览公域作品请前往「发现灵感」。

## 平台集成（Pro）

路径：`/settings/integrations`

### 支持绑定的平台

- 小红书
- 微博
- 微信公众号
- 抖音
- 快手
- 哔哩哔哩

### 绑定步骤

1. 进入平台集成页
2. 查看各平台 **是否已配置**（需运营方在服务端配置 OAuth）
3. 对已开通的平台点击 **连接**
4. 跳转平台授权页，同意授权
5. 回调成功后显示已绑定账号

### 绑定后能做什么

在创作台完成成稿后，可减少「复制 → 打开平台后台 → 粘贴」的步骤，**从有感直接发布**到已绑定平台（与「发布到有感」并列）。

### 未配置时的表现

若某平台显示未配置，说明服务端尚未填入该平台 OAuth 密钥，需联系管理员或等待官方开通。你可先使用「发布到有感」。

### 与会员的关系

平台一键发布为 **Pro / Pro+** 套餐能力。免费版用户请以发布到有感为主。

## 问题反馈

路径：`/feedback`

可提交功能异常、体验建议、使用疑问。提交时会附带页面地址与浏览器信息（不含密码）。正式反馈系统接入前，内容可能复制到剪贴板或通过邮件发送。

## 手机 App 与 Web 同步

- 同一账号登录手机与 Web
- 手机适合**记录灵感与参考**
- 完整定方案、制作、改稿建议在 **Web 创作台** 进行
- App 下载链接受部署环境配置，未配置时页面会提示
